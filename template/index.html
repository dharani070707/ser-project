About Our Project

The recognition of emotions is automatic and unconscious by humans. This is a vital process for people to communicate with each other and thus, to get a better interaction from the human machine, emotions have to be considered. Hence we have developed an application capable of rendering emotion based of the human speech using an Machine Learning model.

Fully Resposive

The website is fully responsive and enables users to play around and have good time adjusting to the environment. The project was made with the view to add more accuracy for the model by adding more dataset enhancement through daily activities.

Clean Code

We have used clean python modules with very less imports as possible and have aimed at achieveing clean and clear code. Model was delveloped in python , front end in HTML, CSS ,JAVASCRIPT. And the integration was done in Flask.

Free of Charge

The project is free of cost for users we will avoid any financing schemes like ads to keep the web application clean and user friendly. Thank you and hope you have a great time Browsing

How This Works?

Record

Record the speech/voice of which the emotion has to be recognized(Avoid any auto-tuning and other noise for better accuracy)

Upload

Upload the file in the form of .wav

Action

Click on the predict emotion button.

Result

An relevent emotion along with an relent emote will be displayed. (consider the fact that the prediction is done by an trained ML model and can be prone to errors)
